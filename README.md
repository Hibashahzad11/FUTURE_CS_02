# Phishing Email Detection & Awareness System

## Overview

This project focuses on identifying phishing emails and spreading cybersecurity awareness. Different email samples were analyzed to detect phishing indicators such as fake senders, suspicious links, urgent language, and grammar mistakes.

## Objectives

* Understand phishing attacks
* Analyze suspicious emails
* Classify emails as safe or phishing
* Educate users about cybersecurity awareness

## Tools Used

* Google Images

## Project Structure

* report.pdf
* email-screenshots/
* README.md

## Conclusion

This project helped improve understanding of phishing attacks and how users can protect themselves from online scams.
